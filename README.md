# Serial-monitor
Serial monitor and plotter for linux

## Screen commands

 First you need to install screen:
```bash

sudo apt install screen

```
Attaching COM port to screen:
```bash
screen /dev/PORT
```

To run screen you type:
```bash
screen
```

Detaching screen:
```bash
Ctrl + a Ctrl + d
```

Listing all opened screens:
```bash
screen -ls
```
